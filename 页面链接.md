# VMS 视频管理系统 - 页面链接导航

> 本地访问地址：`http://localhost`
> 
> 线上访问地址：`https://vms.sikadi.site`（如已配置 Cloudflare Tunnel）
> 
> 📌 前端使用 Hash 路由模式，所有页面链接需要带 `#/` 前缀

---

## 🎬 用户端页面（Portal）

| 页面 | 链接 | 说明 |
|------|------|------|
| **首页** | [http://localhost/#/](http://localhost/#/) | 视频门户首页，热门推荐 |
| **分类页** | [http://localhost/#/category/1](http://localhost/#/category/1) | 按分类浏览视频 |
| **视频播放** | [http://localhost/#/video/1](http://localhost/#/video/1) | 视频详情与播放 |
| **搜索页** | [http://localhost/#/search?keyword=测试](http://localhost/#/search?keyword=测试) | 搜索视频 |
| **观看历史** | [http://localhost/#/history](http://localhost/#/history) | 我的观看历史（需登录） |
| **我的收藏** | [http://localhost/#/favorites](http://localhost/#/favorites) | 收藏的视频（需登录） |
| **个人中心** | [http://localhost/#/profile](http://localhost/#/profile) | 个人信息管理（需登录） |

## 📚 API 文档（Knife4j）

### 🌟 聚合文档（推荐）

| 入口 | 链接 | 说明 |
|------|------|------|
| **统一API文档** | http://localhost:8080/doc.html | 通过网关聚合所有微服务文档，可在一个页面切换查看 |

> ✨ **推荐使用聚合文档**：在左上角下拉菜单中可以切换查看各个微服务的API文档，无需分别访问

### 📖 各服务独立文档

| 服务 | 链接 | 端口 |
|------|------|------|
| **用户服务** | http://localhost/doc/user/doc.html | 8081 |
| **视频服务** | http://localhost/doc/video/doc.html | 8082 |
| **互动服务** | http://localhost/doc/interaction/doc.html | 8083 |
| **运营服务** | http://localhost/doc/operation/doc.html | 8084 |
| **文件服务** | http://localhost/doc/file/doc.html | 8085 |

> 💡 **提示**: 各服务独立文档通过 Nginx 代理访问。如需直接访问某服务文档，使用 `http://localhost:端口/doc.html`
