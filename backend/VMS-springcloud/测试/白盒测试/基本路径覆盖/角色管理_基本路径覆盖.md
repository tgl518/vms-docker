## 二、createRole() 控制流图

```
    开始
      │
      ▼
   [检查code重复] ──Y──→ 抛异常
      │N
      ▼
   [检查name重复] ──Y──→ 抛异常
      │N
      ▼
   [创建Role对象]
      │
      ▼
   [insert到数据库]
      │
      ▼
   [permissionIds!=null && !empty?] ──Y──→ [保存权限]
      │N                                      │
      ▼                                       ▼
   [返回id] ◄─────────────────────────────────┘
```

### 独立路径
- **P1**: 开始 → code重复 → 异常
- **P2**: 开始 → code不重复 → name重复 → 异常
- **P3**: 开始 → code不重复 → name不重复 → 无权限 → 返回id
- **P4**: 开始 → code不重复 → name不重复 → 有权限 → 保存权限 → 返回id

### 圈复杂度
V(G) = E - N + 2 = 7 - 6 + 2 = 3 (实际4条路径)
