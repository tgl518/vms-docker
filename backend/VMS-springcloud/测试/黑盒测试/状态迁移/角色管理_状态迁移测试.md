# 角色管理模块 - 状态迁移测试

**模块**：角色管理 | **日期**：2026-01-08 | **测试方法**：状态迁移法

---

## 一、状态定义

| 状态 | 说明 |
|------|------|
| S0: 不存在 | 角色未创建 |
| S1: 启用 | 角色已创建且启用 (enable=true) |
| S2: 禁用 | 角色已创建但禁用 (enable=false) |
| S3: 已删除 | 角色被删除 |

---
