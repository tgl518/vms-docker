# 角色管理模块 - 等价类划分表

**模块**：角色管理 | **日期**：2026-01-07 | **测试方法**：等价类划分法

---

## 一、创建角色 (POST /user/role)

| 编号 | 输入项 | 有效等价类 | 无效等价类 |
|------|--------|------------|------------|
| EC01 | code | 1-50字符的唯一编码 | 空/null、超50字符、已存在的编码 |
| EC02 | name | 1-100字符的唯一名称 | 空/null、超100字符、已存在的名称 |
| EC03 | description | null 或 1-255字符 | 超255字符 |
| EC04 | sort | null 或 非负整数 | 负数 (可接受) |
| EC05 | enable | null/true/false | - |
| EC06 | permissionIds | null/空列表/有效ID列表 | 无效ID |

---

## 二、更新角色 (PATCH /user/role/{id})

| 编号 | 输入项 | 有效等价类 | 无效等价类 |
|------|--------|------------|------------|
| EC07 | id | 存在的普通角色ID | 不存在的ID、负数ID |
| EC08 | name | 不与其他角色重复 | 与现有角色名重复 |
| EC09 | 系统角色 | 修改普通角色的enable | 禁用SUPER_ADMIN/admin |
| EC10 | 权限 | 普通角色清空权限 | 清空系统角色权限 |

---

## 三、删除角色 (DELETE /user/role/{id})

| 编号 | 输入项 | 有效等价类 | 无效等价类 |
|------|--------|------------|------------|
| EC11 | id | 无用户关联的普通角色ID | 不存在的ID |
| EC12 | 系统角色 | 删除普通角色 | 删除SUPER_ADMIN/admin |
| EC13 | 用户关联 | 无用户关联 | 有用户关联的角色 |

---

## 四、查询角色 (GET /user/role/{id})

| 编号 | 输入项 | 有效等价类 | 无效等价类 |
|------|--------|------------|------------|
| EC14 | id | 存在的角色ID | 不存在的ID、负数ID |

---

## 五、分页查询 (GET /user/role/page)

| 编号 | 输入项 | 有效等价类 | 无效等价类 |
|------|--------|------------|------------|
| EC15 | pageNo | null (默认1) / 正整数 | 0、负数 |
| EC16 | pageSize | null (默认10) / 正整数 | 0、负数 |
| EC17 | name | null / 任意字符串 | - |
| EC18 | enable | null / true / false | - |

---

**统计**：有效等价类 18个，无效等价类 15个，共 33个
